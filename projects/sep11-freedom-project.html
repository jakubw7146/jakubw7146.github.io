Freedom Project

In the Freedom Project I made a 2D browser game using a my tool and javascript. The game is about a guy that has to escape an underground facility by safely going through all the obstacles. I got inspiration from games like Super Mario and Sonic because they are fun 2D games with a pixel art style.

The tool I used was Impact which is basically a javascript game engine that allows you to develop epic HTML5 browser games. You can create custom maps, levels, entities, and sounds and use them in your game. It also has many other features to make your game unique and fun.

Some of my challenges were getting the level editor set up for the first time. However after a lot of brainstorming and innovating I came up with a solution which was to use replit.
